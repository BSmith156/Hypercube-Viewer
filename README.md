# Hypercube Viewer

A Python tool to view n-dimensional hypercubes and rotate them in n-dimensional space.

## Installation

```sh
git clone https://github.com/BSmith156/Hypercube-Viewer.git
```

## Usage

```sh
python hypercube_viewer.py
```
